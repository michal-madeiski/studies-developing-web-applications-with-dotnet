# Developing Web Applications with .NET - laboratory classes
__stage of studies: 3rd year, 5th semester__

Each lab class dir contains an implementation of tasks about the given topic.

* Lab01 - introduction to html and css
* Lab02 - basics of html
* Lab03 - basics of css
* Lab04 - basics of javascript
* Lab05 - introduction to c# and .net
* Lab06 - basics of c#
* Lab07 - linq queries and reflection in c#
* Lab08 - routing and basic views of the razor engine in asp.net
* Lab09 - models, views, layouts and razor engine in asp.net
* Lab10 - entity framework
* Lab11 - razor pages, session and cookies
* Lab12 - identity, api and ajax (implementation in Lab10 dir)
* Lab13 - introduction to angular

_Lab[number] is a one-week homework assignment._
